QWorldとは？

QWorldは、モノイダル圏における図式計算を直感的かつ効率的に描画するために設計されたLaTeXパッケージです。特に、圏論的手法が重要な役割を果たす量子論、トポロジカル量子計算、及び関連分野における図式的推論の可視化を支援します。

本パッケージはTikZを基盤としており、シンプルなコマンドで複雑な図式を容易に描画できることが特徴です。

インストール

QWorldパッケージのスタイルファイル `qworld.sty` は、CTAN上の以下のページから入手可能です:

https://ctan.org/pkg/qworld

ダウンロードしたファイルを TeXのパッケージディレクトリに配置し、プリアンブルに以下を記述することで使用可能となります:

\usepackage{qworld}

TeX LiveやMiKTeXなどのディストリビューションに含まれている場合は、`tlmgr`やMiKTeX Consoleなどを利用してインストールすることもできます。

基本要素

QWorldの最も基本的なコマンドは、`\q{...}`キャンバス内に記述します。これにより、図の素子を配置し、組み合わせていくことができます。

例: ボックスの合成

コード

\q{ \qbox \qcirc \qbox }

`\qbox`でボックスを描画し、`\qcirc`で垂直に合成します。これは数学の関数合成 `g ∘ f` に似ています。

例: ボックスの並置

コード

\q{ \qbox \qbox \qbox }

キャンバス内でコマンドを並べると、描画結果でも要素が水平に並びます。

ボックスのカスタマイズ

`qworld.pdf`の3.3節で解説されているように、キーを使ってボックスの入出力ワイヤーの数や配置を柔軟に調整できます。

例: 複雑な入出力を持つボックス

コード

\q{
  \qbox[
    n=3, N={2},
    s=4, S={1,4},
    hlen=2, vlen=2,
    name={f}
  ]
}
  • `n=3, N={2}`: 上部に3つの接続点を用意し、2番目からワイヤーを出す。
  • `s=4, S={1,4}`: 下部に4つの接続点を用意し、1番目と4番目からワイヤーを出す。
  • `hlen=2, vlen=2`: ボックスの水平・垂直サイズを調整。

モノイダル圏

モノイダル圏は、対象の「積」を表現するためのテンソル積 `⊗` を持つ圏です。QWorldでは、射の並列合成（テンソル積）をコマンドの並置によって自然に表現します。

例: 交換則

コード

% (f₁ ⊗ g₁) ∘ (f₂ ⊗ g₂) と
% (f₁ ∘ f₂) ⊗ (g₁ ∘ g₂) が等しいことを示す
\q{
  \qbox[name={\(f_2\)}]\qbox[name={\(g_2\)}]\n
  \qbox[name={\(f_1\)}]\qbox[name={\(g_1\)}]
}
=
\q{
  \qbox[name={\(f_2 \circ f_1\)}]
  \qbox[name={\(g_2 \circ g_1\)}]
}

この法則は、図式のどの部分を先に計算しても（垂直方向か水平方向か）、結果が変わらないことを保証します。QWorldの構文は、この法則を図として直感的に表現します。

組紐と対称性

組紐モノイダル圏では、ワイヤーを交換するための射 `\qbraid` が導入されます。これにより、ワイヤーの交差を表現できます。

例: ヤン・バクスター方程式

コード

\q{
  \qbraid\qwire\n
  \qwire\qbraid\n
  \qbraid\qwire
}
=
\q{
  \qwire\qbraid\n
  \qbraid\qwire\n
  \qwire\qbraid
}

3本のワイヤーを編む際に、どの2本を先に編んでも最終的な形が同じになることを示す、根本的な関係式です。

双対性

双対性を持つ圏では、ワイヤーを曲げる操作が可能になります。これは `\qcap`（キャップ）と `\qcup`（カップ）コマンドによって実現され、ワイヤーの向きを反転させることができます。

例: ジグザグ等式 (Snake Identity)

コード

\q{
  \qcap\qwireup\n
  \qwireup\qcup
}
=
\q{ \qwireup[vlen=2] }

ワイヤーを下に曲げ（`\qcup`）、すぐに上に戻す（`\qcap`）と、元のまっすぐなワイヤーに戻る、という自明な関係を示します。この「ほどける」性質が、図式計算の強力な基盤となります。
